User Story

-(role) can (action)
-As a (role) I want (feature) so that (benefit).

Fibonacci: 1,2,3,5,8,13,21,34

: As a user, I want to visit a mobile-compatible site from my mobile device.
: As a user, I want to have access to an interactive map when on a desktop to increase interactivity with the application.
: As a user, I want to see a table of states that I can select.
: As a user, I want to click on a state and see a dropdown with its members of congress.
: As a user, I want to click on a different state and have its dropdown expand and all other active dropdowns contract.
: As a user, I want to select a member of Congress and the app displays a visual representation of financial data.
: As a uesr, I want to be able to search members of Congress manually by name without searching through a map/table.
: As a dev, I want to display a precise error message when the user types an invalid search.
: As a user, I want the states to highlight when the cursor hovers over them so that I can tell which state I will click on.
: As a desktop user, I want to be able to toggle between map and table views.
: As a mobile user, I want to be able to interact with the map view when I flip my device into landscape mode.
: As a dev, I want instructions above the map so that users know how to use the application.
